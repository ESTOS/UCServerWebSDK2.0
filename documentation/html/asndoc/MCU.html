MCU - PRIVATE -

Media Control Unit interface

Module description

This module defines structures and operations for controlling a MCU.
The operations are used from the UCServer. It is not used by the clients.
Actually this interface is used by the UCServer to control the Kurento Media Server.

Operations:

Name Description

asnMCUCreatePipeline

- PRIVATE - Create a WebRtcToRTP Relay Channel

asnMCUCreateMediaElement

- PRIVATE - Create an additional Endpoint in an existing MediaPipeline

asnMCUElementProcessAnswer

- PRIVATE - Process sdp Answer for a Element

asnMCUElementRenegotiate

- PRIVATE - Process new sdp Offer for existing endpoint, new endpointID may be generated in Result

asnMCUElementGetStats

- PRIVATE - get Stats or set stats signaling behavior

Events:

Name Description

asnMCUServerStatus

- PRIVATE - WebService reports the MCU Server Status to the UCServer

asnMCUiceCandidate

- PRIVATE - Invoked (Event) signalling a new ice candidate

asnMCURemovePipeline

- PRIVATE - Remove a MediaElement or if no elementID given the pipeline

asnMCUError

- PRIVATE - Invoked (Event) signalling a MCU Error message

asnMCUElementStatsEvent

- PRIVATE - Invoked (Event) signalling a statistics message
