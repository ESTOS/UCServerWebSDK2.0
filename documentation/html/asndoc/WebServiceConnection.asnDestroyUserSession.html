asnDestroyUserSession

Destroy a user session in the UCServer

This function has the same argument as asnNegotiateServerConnection
A user login session object (ENetCtiConnection) is created in the UCServer and ready for asnLogon

ArgumentType: AsnDestroyUserSessionArgument

- PRIVATE -

ResultType: AsnDestroyUserSessionResult

- PRIVATE -

AsnDestroyUserSessionResult is an Object

Name Type Optional Comment

ErrorType: AsnRequestError

AsnRequestError is an Object

Name Type Optional Comment

iErrorDetail

number

A number representing the error

u8sErrorString

string
