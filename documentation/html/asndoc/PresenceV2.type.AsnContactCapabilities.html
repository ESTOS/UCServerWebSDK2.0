AsnContactCapabilities

Structure of client capabilities.
This structure has been superseeded by the AsnClientCapabilitiesV2 structure

- DEPRECATED -

AsnContactCapabilities is an Object

Name Type Optional Comment

bAudio

boolean

At least one client has audio capabilities.

bVideo

boolean

At least one client has video capabilities (camera).

bScreenShare

boolean

At least one client has screen sharing capabilities.

bExchangeCalendar

boolean

At least one client has calendar replication capabilities for Microsoft Exchange.

bNotesCalendar

boolean

At least one client has calendar replication capabilities for Notes.

bCamControl

boolean

At least one client makes it possible that his camera can be remote controlled (left, right, up, down, zoom in/out).
