AsnACUCheckUpdateNowResult

- PRIVATE -

AsnACUCheckUpdateNowResult is an Object

Name Type Optional Comment

bUpdateNow

boolean

true if the client is allowed to install the update now, false if not

iMaxWaitTimeInMinutes

number

If specified, this is the configured maximum Wait time the client has to wait until it forces the update.
If this value is 0 the client must wait until bUpdateNow gets true
If this value is greater than 0 the client must store the timestamp it first received it.
