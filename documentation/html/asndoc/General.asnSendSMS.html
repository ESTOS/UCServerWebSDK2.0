asnSendSMS

Mit dieser Funktion kann ein Client SMS verschicken.
Der Server antwortet Synchron nur generierten ReferenzIDs für jede SMS, die dann
dem später folgenden Event zugeordnet werden kann. Damit können im Falle mehrerer
SMS der Zustand einzeln verfolgt werden.

ArgumentType: AsnSendSMSArgument

- PRIVATE -

AsnSendSMSArgument is an Object

Name Type Optional Comment

messages

AsnSMSMessages

Liste der SMS die verschickt werden sollen

ResultType: AsnSendSMSResult

- PRIVATE -

AsnSendSMSResult is an Object

Name Type Optional Comment

u8sCrossRefIDs

UTF8StringList

CrossrefIDs für die Nachrichten in der gleichen Reihenfolge

ErrorType: AsnRequestError

AsnRequestError is an Object

Name Type Optional Comment

iErrorDetail

number

A number representing the error

u8sErrorString

string

A string representing the error
