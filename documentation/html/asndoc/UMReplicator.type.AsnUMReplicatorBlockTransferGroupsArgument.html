AsnUMReplicatorBlockTransferGroupsArgument

REQUEST: asnUMReplicatorBlockTransferGroups
Funktion startet Transaction, added/updated/deleted die Gruppen in einem Block und
committet in einem. Fehler werden im Server abgehandelt und entsprechend der Rollback
ausgelöst. Der Replikator-Client bekommt dann nur noch das Ergebnis.
