asnRankingRegisterSessionAgent

setzt den preferierten Agenten für die laufende Client-Session

Legt den für die Session aktiven Agenten fest oder gibt ihn wieder frei.
Die Angabe eines neuen Agenten verwirft automatisch den bisherigen.
Sobald die Client-Session beendet wird, ist der Agent automatisch aus dieser Session abgemeldet
