AsnDeletedItem

A deleted item - contact or AsnClientContactItem.

If an element of the client content is deleted, this object contains the ID of the object.
All fields are optional and only be set if necessary.

If a complete cluster has to be removed, only the cluster ID is given.

If a complete group has to be removed, the group and the according cluster ID is given.

If an item has to be removed, the item, gourp and cluster ID is given.

If a contact has to be removed, its contact ID is given.

It is possible that all four fields are set, when the Item and the contact attached to should be removed.
