AsnConfigurationManagerConfiguration

mehrere ConfigurationSettings sind eine Configuration

- PRIVATE -

AsnConfigurationManagerConfiguration is an Array of AsnConfigurationManagerConfigurationSetting

AsnConfigurationManagerConfigurationSetting is an Object

Name Type Optional Comment

u8sKey

string

Ein Key ist hierarchisch aufgebaut (und kein "." am Ende)
bspw. "general.xmpp.ProxyEnabled"

u8sObjectName

string

falls der Key ein benanntes Objekt repraesentiert, dann ist dies der Name
bspw. "default" (bspw. bei einem Standort)

u8sHomeServer

string

leer, falls das Setting Teil der Basiskonfiguration sein soll.
Oder der Name des einzigen UCServers, fuer den
dieses (von der Basiskonfiguration abweichendes) Setting gelten soll.

u8sValue

string

der Wert (beliebiger String)

u8sType

string

eines der Strings: TEXT, NUMBER, FLOAT, BOOLEAN, JSON
(TODO: enumeration machen)

u8sWrittenByServer

string

der Name des Servers, der dieses Setting in die DB geschrieben hat
kann beim Senden zum Webservice leer gelassen werden
wird vom Webservice gesetzt

iTriggerUpdate

number

soll die Aenderung dieses Settings einen Update an die anderen UCServers triggern?
