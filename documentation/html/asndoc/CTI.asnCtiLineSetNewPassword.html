asnCtiLineSetNewPassword

Set a password for a line.

On some PBX systems (especial cloud based) a user has to have a password to access a phone line.
To support this feature, a client can use this function to set the password, so the UCServer can monitor the line.

ArgumentType: AsnCtiLineSetNewPasswordArgument

- PRIVATE -

AsnCtiLineSetNewPasswordArgument is an Object

string
