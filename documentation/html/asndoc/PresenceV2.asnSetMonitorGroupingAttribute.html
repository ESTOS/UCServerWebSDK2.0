asnSetMonitorGroupingAttribute

Set attributes for possible grouping in the client monitor.

The grouping attribute (or main attribute) of a contact gives the user the possibility to arrange contacts in the clients monitor
by skills or other defineable texts.

The admin of the UCServer can define a set of attributes a user can select one for himself, which is then dispatched to other users in the presence system.
All users with the same attribute set can then be grouped together in the favorites monitor of the client.

A attribute consists of a attribute string and a string with a corresponding abbrevation of it (short form).
It is possible to set a free text for an additional, user defined attribute, which can also be displayed in the client.
