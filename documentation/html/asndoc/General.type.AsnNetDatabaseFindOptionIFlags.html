AsnNetDatabaseFindOptionIFlags

- PRIVATE -

AsnNetDatabaseFindOptionIFlags is an Enumeration

Name Type Value Comment
eRAWNAMEDATA number 1 Alt: Wird nicht mehr benötigt! // Im Feld "m_strLastName" kann sich eine Komination aus Vor- und Nachnamen befinden, "m_strFirstName" wird ignoriert
ePHONEBOOKS number 2 Telefonbücher in Suche einbeziehen
eOTHERDBS number 4 Restliche, hier bisher nicht genannte Datenbanken in Suche einbeziehen
eRIGHTSCHECKED number 8 Wurde der Benutzer bereits überprüft, ob er die nötigen Berechtigungen hat
Wenn das Flag gesetzt ist, wurde der Benutzer bereits überprüft
ePHONETICAL number 16 Phonetische Suche anschalten
eMETADIRECTORY number 32 MetaDirectory mit in die Suche einbeziehen
eSINGLENUMBER number 64 Tel-Buch-Einträge mit mehreren Nummern werden nicht zusammengefasst, sondern einzeln gehalten. Die Telefonnummer steht in "m_strPhoneBusiness"
eFORCEMOBILE number 128 Die Suche nach Mobilrufnummern bei TVG zulassen, auch wenn kein Ort angegeben wurde (zeitintensiv!!)
eMULTIFIELDNAME number 256 nur der LastName enthält einen Suchstring, der in mehreren Feldern gesucht wird (auch in Company...) die anderen Suchfelder sollten leer bleiben
eONLYCTISERVERUSERS number 512 Nur CtiServer User suchen (betrifft nur NachCache im CtiServer)
eMANUALSEARCH number 1024 Die Suche wurde manuell vom Benutzer gestartet und ist keine Suche, die durch einen Call automatisch angestoßen wurde
eSORTBYDISPLAYNAME number 2048 Die Suchergebnisse innerhalb des Suchknotens (!) nach dem Anzeigenamen sortiert zurückgeben
eUSENOCACHE number 4096 Immer neu suchen
eCOMPANYSEARCH number 8192 Suche nach Firmenrufnummern (ignoriere Durchwahlen)
eNOPHONEBOOKS number 16384 Telefonbücher NICHT in Suche einbeziehen
eMANDANTSEARCH number 32768 Suche nach gerufener Nummer (Mandantensuche, Hotline-Funktion)
eONLYMETADIRECTORY number 65536 Nur im MetaDirectory suchen
eJPEGPHOTOHASH number 131072 Wenn nicht gesetzt, werden die Bild-Binärdaten direkt im optionalParam 'jpegPhoto' übertragen. Andernfalls wird nur der optionalParam 'jpegPhotoHash' übertragen, anhand dem mittels 'asnRequestContactImage' die Binärdaten separat abgeholt werden können. Funktioniert nur bei UCServer Benutzern.
