AsnSubscribeClientContentFlags

Flags how to subscribe on client content.

AsnSubscribeClientContentFlags is an Enumeration

Name Type Value Comment
eALL number 0 eALL = eCONTACTS | eSTRUCTURE | eJPEGPHOTO
eCONTACTS number 1 Get events for changes in the contacts.
eSTRUCTURE number 2 Get events for changes in the structure.
eJPEGPHOTO number 4 If set, the result includes all existing contact photos. If not set the result contains only the hash values of the photos, and the client can decide if it wants to fetch the photos with asnRequestContactImage.
