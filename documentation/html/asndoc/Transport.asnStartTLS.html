asnStartTLS

Start TLS Handschake on existing connection

Client should use this function immediately after establishing the connection.
After calling this function, the UCServer switches to TLS mode and expects a regular TLS handshahe from the client.

ArgumentType: AsnStartTLSArgument

AsnStartTLSArgument is an Object

Name Type Optional Comment

ResultType: AsnStartTLSResult

AsnStartTLSResult is an Object

Name Type Optional Comment

ErrorType: AsnRequestError

AsnRequestError is an Object

Name Type Optional Comment

iErrorDetail

number

A number representing the error

u8sErrorString

string
