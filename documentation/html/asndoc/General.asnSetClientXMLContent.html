"NewClientContentRevision" (UTF8String):
If Save was successful, it will return 'NewClientContentRevision' in optional Parameters, which contains
the new Revision which must be saved by the Client for the next call to asnSetClientXMLContent.
(Introduced with PCE 4.1)

optionalParams Name "ClientContentOutOfDate" Value number
Will be returned if the attempt failes because the server already has a newer version of the client content. Update the client content to the actual version is recommended. The value can be ignored and is always 0xffffff02 (CLIENTCONTENT_XML_OUTOFDATE). (Introduced with PCE 4.1)
optionalParams Name "ClientContentXMLError" Value number
Will be returned if the attempt failes because of a XML-Error in the transmitted XML-String. The value can be ignored and is always 0xffffff01 (CLIENTCONTENT_XML_ERROR). (Introduced with PCE 4.1)
optionalParams Name "FileErrorCode" Value number
Will be returned if the XML-String can't be saved on the server because of a DOS error. The Value contains the Error Code described in http://msdn.microsoft.com/en-us/library/t3ayayh1(VS.80).aspx (Introduced with PCE 4.0.2)
