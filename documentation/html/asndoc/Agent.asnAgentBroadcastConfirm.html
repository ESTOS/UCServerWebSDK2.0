asnAgentBroadcastConfirm

Confirms a received broadcast

The agent client confirms to the server that it is continuing to process the broadcast data.
If the TTL for the broadcast has not expired by then, the associated broadcast item is removed from the list of open (unanswered) elements by the server.

ArgumentType: AsnAgentBroadcastConfirmArgument

- PRIVATE -

AsnAgentBroadcastConfirmArgument is an Object

Name Type Optional Comment

u8sBroadcastID

string

The BroadcastID generated by the server. The server ignores duplicate confirmations! See AsnAgentBroadcastCreateArgument.
