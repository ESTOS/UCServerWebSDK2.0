asnClientContentTaggedIDsChanged

Event from server to client if tags on contacts have changed.

If another client changes the tag list, this event will be fired to all clients with the new list.

ArgumentType: AsnClientContentTaggedIDsChangedArgument

AsnClientContentTaggedIDsChangedArgument is an Object

Name Type Optional Comment

u8sTaggedIDs

UTF8StringList

New contact ID list of the contacts to be tagged.
