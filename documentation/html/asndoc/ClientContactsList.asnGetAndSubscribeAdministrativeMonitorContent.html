asnGetAndSubscribeAdministrativeMonitorContent

Client gets the contacts especial for a attendant console and can subscribe to events if the groups are changing.

For fast call transfer or agent consulting it must be possible to get contact for this purpose, because that
enables additional, specialized features for them in the UI for this use case.

The UCServer will gather the groups and the user IDs according the configuration set by the admin.

ArgumentType: AsnGetAndSubscribeAdministrativeMonitorContentArgument

AsnGetAndSubscribeAdministrativeMonitorContentArgument is an Object

Name Type Optional Comment

iEventFlagsToSubscribe

number

yes

Optional parameter. If left out the UCServer assume eAdministrativeMonitorContentGroupChanges (1) as default.
This parmameter can be set to eAdministrativeMonitorContentGroupNoEvents (0) for requesting the actual content list but not to subscribe to events. (Linked Type: AsnAdministrativeMonitorContentSubscriptionFlags)

AsnGetAndSubscribeAdministrativeMonitorContentResult is an Object

List of AsnAdministrativeMonitorGroup objects the clients user is configured for on the UCServer.
