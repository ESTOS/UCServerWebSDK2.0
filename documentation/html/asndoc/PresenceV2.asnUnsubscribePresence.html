asnUnsubscribePresence

Operation to end subscriptions on contacts on the ucserver.

To stop getting events from contacts on the ucserver this operation is called with the list contact IDs of these contacts.

If the operation is executes successful, it will return the result AsnUnsubscribePresenceResult

If an error occured, AsnRequestError will be returned.

ArgumentType: AsnUnsubscribePresenceArgument

AsnUnsubscribePresenceArgument is an Object

Name Type Optional Comment

seqContactIDs

UTF8StringList

List of contact IDs to unsubscribe from (e.g. 'sip:tester@test.com').

ResultType: AsnUnsubscribePresenceResult

AsnUnsubscribePresenceResult is an Object

Name Type Optional Comment

iResult

number

ErrorType: AsnRequestError

AsnRequestError is an Object

Name Type Optional Comment

iErrorDetail

number

A number representing the error

u8sErrorString

string

A string representing the error
