asnDeleteJournalEntries

Delete journal entries form the database.

Depending on the persmissions, a client can delete journal entries of the user or from the groups. If the call fails, AsnRequestError will be used instead of the result.

ArgumentType: AsnDeleteJournalEntriesArgument

AsnDeleteJournalEntriesArgument is an Object

Name Type Optional Comment

u8sConnectionIDList

UTF8StringList

List of connection ids to indentify the journal entries to delete

ResultType: AsnDeleteJournalEntriesResult

AsnDeleteJournalEntriesResult is an Object

Name Type Optional Comment

iResult

number

0 if the call was successful

ErrorType: AsnRequestError

AsnRequestError is an Object

Name Type Optional Comment

iErrorDetail

number

A number representing the error

u8sErrorString

string

A string representing the error
