asnWebServiceCapabilitiesChanged

Webservice reports changed capabilities to the UCServer

Whenever the capabilities of the webservice changes, it reports the changes to the UCServer.-
For example when the web service established a connection to the kurrento server it reports that it now can handle MCU commands.

ArgumentType: AsnWebServiceCapabilitiesChangedArgument

- PRIVATE -

AsnWebServiceCapabilitiesChangedArgument is an Object
