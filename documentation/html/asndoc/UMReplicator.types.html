UMReplicator - PRIVATE -

Usermanagment replication interface

Module description

This module defines structures and operations for the user management replication.
The replication is done by an external command line tool (UMReplicator.exe) and is used to transfer
LDAP accessible user/group/computer data into the user management of the UCServer.
