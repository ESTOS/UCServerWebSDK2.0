AsnNetDatabaseJournalFilterFlags

Flags for filtering journal entries in the result list of asnGetJournalEntries These are used in AsnNetDatabaseJournalFindOptions

This Structure is a BIT STRING, means that every bit in this string has a "name" (== number with a symbolic name in source code).

In JSON encoding we transfer that as a string of 0 and 1 characters; see the values for the corresponding string.
To set more than one bit, the corresponding character should be a "1".

Example: "0011000001" => Inbound + Outbound + IncludeAllPublic

Since this is not very handy to use, a optionalParam "iAdditionalFlags" was introduced, where this flags can be set in the more
convenient way of Bits using normal Bit-logic. See AsnNetDatabaseJournalFilterEnumeration for the Bits to set/read.
