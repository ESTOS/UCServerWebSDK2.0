asnCtiNotifyLineInfoChanged

Event from the server if a line changed its status or has been added or removed.
In order to get these events the line must be monitored with asnCtiLineMonitorStart

ArgumentType: AsnCtiNotifyLineInfoChangedArgument

AsnCtiNotifyLineInfoChangedArgument is an Object

Name Type Optional Comment

u8sMonitorCrossRefID

string

CrossrefID for the corresponding asnCtiLineMonitorStart.

u8sLinePhoneNumber

string

Line identifier for the line where the changes happened.

lineInfo

AsnCtiLineInfo

The new line information.
