AsnConfigurationManagerConfigurationOpStatusList

Liste mehrerer operation statuses

- PRIVATE -

AsnConfigurationManagerConfigurationOpStatusList is an Array of AsnConfigurationManagerConfigurationOpStatus

AsnConfigurationManagerConfigurationOpStatus is an Object

Name Type Optional Comment

u8sKey

string

Name der Operationseinheit im UCServer
Ein Key ist hierarchisch aufgebaut (und kein "." am Ende)
bspw. "OpStatus.smsmanager"

u8sOpName

string

Name der Operation

u8sAdditionalInfo

string

zusaetzliche Infos e.g. about the arguments

u8sHomeServer

string

Name des UCServers, um dessen Operationseinheit es hier geht

iStatus

number

Status (Fehlercode) der beschriebenen Operation
Interpretation des Fehlercodes ist operationsspezifisch
(0 ist normalerweise no error)

u8sStatus

string

Text fuer den Fehlercode
spezifisch fuer die Operation
