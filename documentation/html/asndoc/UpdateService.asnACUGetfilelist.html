asnACUGetfilelist

Client submits which software is installed and get a list of files to download

After processing the SoftInfoLocation List from asnACUGetSoftInfoLocations, the client tells the server,
which software is installed on the client.
The result is a list of files that should be downloaded and processed by the Update Service on the Client.
