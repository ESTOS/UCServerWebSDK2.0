öffentliche Präsenz eines Agenten

Die öffentliche Präsenz eines Agenten ermittelt sich aus der aktuellen Präsenz des ActiveContact sowie dem jeweiligen Agentenstatus innerhalb der Agenten-Gruppe.

AsnRankingAgentPublicPresenceEnum is an Enumeration

Name Type Value Comment
offline number 0 Agent ist offline / abgemeldet
uncontrolled number 1 ggf. kein Agent?
available number 2 Agent ist angemeldet und verfügbar
notavailable number 3 Agent ist angemeldet aber beschäftigt
