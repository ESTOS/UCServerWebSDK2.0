asnLogonV2

Used to login to the UCServer
This function supersedes the former asnLogon which was using a custom Error object
The new implementation uses the standard AsnRequestError and puts renegotiation into the AsnLogonResultChoice object

ArgumentType: AsnLogonArgument

AsnLogonArgument is an Object

Name Type Optional Comment

iVersion

number

version of the protocol

u8sVersion

string

Client Version (z.B. 3.0.0.100)

iAttachToContact

number

Event notification Option (Linked Type: AsnAttachToContactOptions)

u8sClientComputerName

string

Computer Name - for Display Purposes

authentication

AsnAuthenticationChoice

Authentication Options

optionalParams

AsnOptionalParameters

yes

optionalParams Name "CompletedEventID" Value string

ResultType: AsnLogonResultChoice

AsnLogonResultChoice is a Choice

Name Type Comment
success

AsnLogonResultSuccess

 Logon has succeeded the client receives the result of the operation
renegotiation

AsnAuthenticationSimpleCrypt

 Logon requires a renegotiation

ErrorType: AsnRequestError

AsnRequestError is an Object

Name Type Optional Comment

iErrorDetail

number

A number representing the error

u8sErrorString

string

A string representing the error
