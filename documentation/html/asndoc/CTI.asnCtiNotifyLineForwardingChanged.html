asnCtiNotifyLineForwardingChanged

Event from the server if a line changed its forwardings.
In order to get these events the line must be monitored with asnCtiLineMonitorStart

ArgumentType: AsnCtiNotifyLineForwardingChangedArgument

AsnCtiNotifyLineForwardingChangedArgument is an Object

Name Type Optional Comment

u8sMonitorCrossRefID

string

CrossrefID for the corresponding asnCtiLineMonitorStart.

u8sLinePhoneNumber

string

Line identifier for the line where the changes happened.

seqLineForwards

AsnLineForwards

New forwarding settings on the line.
