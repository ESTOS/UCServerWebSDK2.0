asnResponseSetServiceAgentStatesFailed

Event to for agent states that the UCServer was unable to set

If the UCServer was unable to set agent states while processing asnSetServiceAgentStates, it will
send back this event with the failed agent states.

ArgumentType: AsnResponseSetServiceAgentStatesFailedArgument

AsnResponseSetServiceAgentStatesFailedArgument is an Object

listServiceAgentStates

AsnServiceAgentStateFailedReasonList

List of AsnServiceAgentStateFailedReason.
