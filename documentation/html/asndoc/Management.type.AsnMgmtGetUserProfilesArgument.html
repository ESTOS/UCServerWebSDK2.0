AsnMgmtGetUserProfilesArgument

get all user profiles

Create a new GUID and send it as parameter u8sCrossRefId
UCServer will then send Events asnMgmtGetUserProfilesResult with this GUID

AsnMgmtGetUserProfilesArgument is an Object

Name Type Optional Comment

iActiveUsersOnly

number

CTI-Enabled users only

iFilterOnlineState

number

Filter with actual onlinestate -> AsnAdminUserOnlineStateFilter

u8sCrossRefId

string
