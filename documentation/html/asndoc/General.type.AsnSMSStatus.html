AsnSMSStatus

Der Server schickt dem Client Status-Nachrichten zu den versendeten SMS

- PRIVATE -

AsnSMSStatus is an Enumeration

Name Type Value Comment
eSMSSTATUSSUCCESS number 0 SMS erfolgreich versendet
eSMSSTATUSERROR number 1 SMS konnte nicht übermittelt werden
eSMSSTATUSDELAYED number 2 SMS wird später verschickt (z.B. retry)
