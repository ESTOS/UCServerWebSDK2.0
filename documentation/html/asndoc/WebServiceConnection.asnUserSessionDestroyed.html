asnUserSessionDestroyed

Notification, that a user session has been destroyed in the UCServer.

This Event is sent by the UCServer when a user session has been destroyed.
This may be because of a timeout, missing keepAlive or the client called asnLogoff

ArgumentType: AsnUserSessionDestroyedArgument
