asnGetJournalEntriesCountV2

Count journal entries in the database ang get the result.

A client can get the count of journal entries from the server in the database for a given filter named 'findOptions' in the argument.

This can be used for displaying the count of such journal entries without transferring a lot of journal data via the network.
Popups or pins with the count of unread entries are examples for this.

ArgumentType: AsnGetJournalEntriesCountV2Argument

AsnGetJournalEntriesCountV2Argument is an Object

Name Type Optional Comment

findOptions

AsnNetDatabaseJournalFindOptionsV2

specify the journal entries to count

ResultType: AsnGetJournalEntriesCountResult

AsnGetJournalEntriesCountResult is an Object

Name Type Optional Comment

iCount

number

Count of journal entries found in the database according the AsnNetDatabaseJournalFindOptions given in the argument
