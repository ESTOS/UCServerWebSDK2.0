AsnAgentStateEnum

The possible states the agent can have.

This is the state of the agent; hence it is an integer and not a bit field.
This value is to be used for server messages as well as for setting these states.

AsnAgentStateEnum is an Enumeration

Name Type Value Comment
eAGENTSTATENOAGENT number 0 Agent logged out from group - (is set implicitly at AgentLogoff() by the server)
eAGENTSTATENOTREADY number 1 Agent unavailable
eAGENTSTATEREADY number 2 Agent available in group - (is initially set at AgentLogin() by the server)
eAGENTSTATEBUSY number 3 Agent is busy
eAGENTSTATEWORKINGAFTERCALL number 4 Agent is working after a call
