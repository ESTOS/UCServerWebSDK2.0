AsnJournalSubscribeFlags

Options for asnJournalSubscribeEvents which defines if the client is interested in only user owened journal entries or it should include the
journal entries of his groups also.

AsnJournalSubscribeFlags is an Enumeration

Name Type Value Comment
eDEFAULT number 0 Get only user owened jounral entries
eGROUPJOURNAL number 1 Include the journal entries of the users groups, if the view permissions in the groups are set accordingly
