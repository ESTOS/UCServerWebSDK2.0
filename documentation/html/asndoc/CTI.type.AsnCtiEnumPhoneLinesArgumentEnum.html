AsnCtiEnumPhoneLinesArgumentEnum

Which lines shall be listed for the user (own, additional lines, all).

AsnCtiEnumPhoneLinesArgumentEnum is an Enumeration

Name Type Value Comment
userOwnLines number 0 Own lines which the logged in user is owner of.
userMonitorLines number 1 Own lines plus the lines, the user has special access rights for.
systemLocalLines number 2 All lines that are local on the system.
systemGlobalLines number 3 All local lines plus all global known lines; for future use, current its the same as systemLocalLines.
