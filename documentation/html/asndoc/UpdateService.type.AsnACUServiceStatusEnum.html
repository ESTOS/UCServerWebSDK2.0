AsnACUServiceStatusEnum

Update Status Values

The Client Update Service reports it status with asnACUOnSetNewStatus
The iNewStatus member contains two statuses.
One in the lower 8 bits (aCUSERVICESTATUSMASK) and one in the higher 8 bits(aCUUPDATESTATUSMASK).
The aCISERVICESTATUS values tell the status of the Update Service.
The aCUUPDATESTATUS values tell the status of an update package.
