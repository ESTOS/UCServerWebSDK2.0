UCServer Client Access Protocol (Preview)

This is the documentation for the UCServer Client Access Protocol (UCAP). UCAP is a protocol for accessing the UCServer services. UCServer is a service that provides access to a variety of services, including presence, chat, CTI functionality, realtime communication via WebRTC and more. UCAP is a protocol that allows clients to access these services using an ASN.1 structured interface. This documentation provides an overview of the UCAP protocol, as well as detailed information on the various services provided by UCServer.
