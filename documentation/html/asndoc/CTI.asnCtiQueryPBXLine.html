asnCtiQueryPBXLine

Find lines in the server for a supercanonic number.

In order to find lines for a given supercanonic number the client can use this function. If the server itself finds lines they are returned.
If the server does not find lines it queries all ECSTA instances if one of them is able to provide a line with this number.
(You need to properly configure this behaviour in line group settings within the ucserver administrator)
