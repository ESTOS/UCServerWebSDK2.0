asnMeshTwoWayAuthenticatePart2

Two way authentication process part2
called from server in client role after part1 was successfully done

  1. we send SHA-1 hash of concatenation of password and u8sSeedServer(sent with AsnMeshTwoWayAuthenticatePart1Result)
    and login data
  2. remote server tests meshid and hashs

ArgumentType: AsnMeshTwoWayAuthenticatePart2Argument

- PRIVATE -

AsnMeshTwoWayAuthenticatePart2Argument is an Object

Name Type Optional Comment

u8sMeshID

string

Mesh ID (Mesh group ID)

u8sPasswordHash

string

Mesh Password SHA-1 hashed with u8sSeedServer

u8sMeshNodeID

string

Calling Mesh Node ID

u8sComputerNameFQDN

string

Calling Mesh Node Name FQDN (DNS-Name)

iListenPort

number

Calling server client listen interface port

u8sClientVersion

string

Calling server version

iCertChainVerifyReturn

number

we tell remote ucserver if we could verify its certificate (yes: 0, no: error number of startTLS function)
