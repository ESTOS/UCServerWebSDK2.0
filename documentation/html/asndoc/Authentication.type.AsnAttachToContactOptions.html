AsnAttachToContactOptions

- DEPRECATED -

AsnAttachToContactOptions is an Enumeration

Name Type Value Comment
eAttachToNothing number 0 Commandline tools: dont attach to anything, just get rights to call something
eAttachToAll number 1 Normal Client Login with attach to the 'old' Eventing Sessions, register for Journal etc.
eAttachOnlyMinimum number 2 Normal Client Login, but don't register for events or other Services; this will be done explicitly
