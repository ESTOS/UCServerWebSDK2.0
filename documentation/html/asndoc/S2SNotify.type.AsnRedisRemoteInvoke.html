AsnRedisRemoteInvoke

This is an exteded version of ROSEInvoke that is used for passing an invoke received from a client to another UCServer via redis

IMPORTS
ROSEInvoke FROM SNACC-ROSE
;
wird nur zwischen 2 ucservern ueber redis verschickt.
Die Antwort ist ein AsnRemoteInvokeCompleteArgument aus der ENetRoseInterface, das auch zum Client weitergeschickt werden kann
