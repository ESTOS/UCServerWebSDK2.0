AsnNegotiateServerConnectionResult

- DEPRECATED -

AsnNegotiateServerConnectionResult is an Object

Name Type Optional Comment

iServerProtocolVersion

number

Protokoll Version des Servers

iServerUMSecurityOption

number

Verfügbare Login Option

iServerTLSSecurityOption

number

TLS nie verwenden, ermöglichen oder voraussetzen.

sServerAuthentication

string

Timestamp des Servers für Verschlüsselung am Client

optionalParams

AsnOptionalParameters

yes

Liste der möglichen optionalen Parameter:

  • optionalParams Name "serverClientWriteablePassword" Value AsnInt definiert ob der Client das PAsswort selbst ändern kann oder nicht (LiveChat Passwort ändern)
  • optionalParams Name "serverVersion" Value String Versionsstring des Servers
  • optionalParams Name "productVersion" Value String UCServer-ProductVersion (für ProCall identisch mit der FileVersion, ProCallGo fängt bei 1.xx an)
  • optionalParams Name "ServerProtocolVersion" Value AsnInt (completly senseless as it is already contained in the iServerProtocolVersion)
  • optionalParams Name "ServerInterfaceVersion" Value AsnInt (an additional internal version (410))
  • optionalParams Name "serverConnectionId" Value String GUID der Verbindung, die der Server zugewiesen hat und dazu verwendet auf diese Verbindung zu verweisen bzw. zu identifizieren.
  • optionalParams Name "presenceDomains" Value String Liste der PrsenceDomains die der Server verwaltet duch ; getrennt.
  • optionalParams Name "PresenceServicesDisabled" Value AsnInt (only if presence is disabled)
  • optionalParams Name "UserUserRightsDisabled" Value AsnInt (only if userrights are disabled)
  • optionalParams Name "AnonymousUsername" Value String - UserName for the Anonymous login
  • optionalParams Name "AnonymousToken" Value String - Token for the Anonymous login
  • optionalParams Name "ServerType" Value AsnInt AsnNegotiateServerConnectionServerType
  • optionalParams Name "ServerManType" Value AsnInt OEMId des Servers (ab Version 6.2 verfügbar)
  • optionalParams Name "ServerBusyTimeout" Value AsnInt Sekunden bis zum nächsten Verbindungsversuch
  • optionalParams Name "Auth_%s" Value AsnString different Authentication methods the server is offering
  • optionalParams Name "localUCWeb" Value AsnString the local ucweb url
  • optionalParams Name "localUCWebs" Value AsnString the local ucweb url
  • optionalParams Name "ucsid" Value AsnString the ucsid of the server
  • optionalParams Name "ucscontroller" Value AsnString the ucscontroller url
  • optionalParams Name "ucsurl" Value AsnString the complete url to contact the controller

stServerTime

string

yes

Timestamp zum Abgleich der Uhrzeit zwischen Client und Server. Ermöglicht z.B. korrekte Anzeige von Call-Duration bei abweichender Serverzeit.
