asnMgmtDatabaseUpdateInProgress

Check if Database Update is Running

This function is used to check if a Database Update is running in background after Server Start
Progress is reported back to Admin with OPID_asnMgmtUpdateDatabaseProgress Events

ArgumentType: AsnMgmtDatabaseUpdateInProgressArgument

AsnMgmtDatabaseUpdateInProgressArgument is an Object

Name Type Optional Comment

ResultType: AsnMgmtDatabaseUpdateInProgressResult

AsnMgmtDatabaseUpdateInProgressResult is an Object

Name Type Optional Comment

bResult

boolean

ErrorType: AsnRequestError

AsnRequestError is an Object

Name Type Optional Comment

iErrorDetail

number

A number representing the error

u8sErrorString

string

A string representing the error
