asnRankingRankSessionAgent

übergibt den aktuellen Ranking-Wert für den Session-Agenten

legt das für den SessionAgent aktuelle Ranking/Punkte fest.

ArgumentType: AsnRankingRankSessionAgentArgument

- PRIVATE -

AsnRankingRankSessionAgentArgument is an Object

Name Type Optional Comment

iPoints

number

absoluter aktueller Ranking-Wert für den Agenten (für diese Session)

ResultType: AsnRankingRankSessionAgentResult

- PRIVATE -

AsnRankingRankSessionAgentResult is an Object

Name Type Optional Comment

u8sSessionAgentURI

string

URI des aktuellen Session-Agenten

iCurrentPoints

number

aktueller Punkte-Wert des Agenten für diese Session

iCumulatedAgentPoints

number

aktueller Ranking-Wert für den Agenten (absolut - kumuliert über alle Connections)

u8sSessionGroupID

string

yes

GroupID des aktuellen Session-Agenten

ErrorType: AsnRequestError

AsnRequestError is an Object

Name Type Optional Comment

iErrorDetail

number

A number representing the error

u8sErrorString

string

A string representing the error
