These are the possible error values in AsnRequestError

AsnBinaryTransferRequestErrorEnum is an Enumeration

Name Type Value Comment
noerror number 0 no error
serverBusy number 1 server is currently serving to many downloads, please try again later
extensionNotAllowed number 2 the file type (extension) is not allowed for file transfer

unknownTransferID number 4 the server does not know this transferID (may occur in upload and download)

invalidPosition number 6 the client was requesting a position which is not within the file beeing downloaded
filedeleted number 7 the file was already deleted
invalidMetaData number 8 the metaData is not allowed or invalid (may occur when initiating upload)
databaseFail number 9 either the connection or read/write from server database failed (see server log for details)
