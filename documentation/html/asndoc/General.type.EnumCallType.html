EnumCallType

Technical type of a call.

- DEPRECATED -

EnumCallType is an Enumeration

Name Type Value Comment
eCALLTYPENORMAL number 0 Just a normal call.
eCALLTYPECONSULT number 1 Consultation call.
eCALLTYPECONFERENCECONTROLER number 2 Conference controller call.
eCALLTYPECALLBACK number 3 Callback call (callback while busy or callback was not answered)
eCALLTYPECONFERENCEMEMBER number 4 Conference member call.
eCALLTYPECONFERENCECONTROLERPENDING number 5 Conference controller while building up the conference (US conference model)
eCALLTYPECONFERENCEMEMBERPENDING number 6 Conference member while building up the conference (US conference model)
