asnCtiNotifyLineInvalidPasswordChanged

Event from the server if the password state on the line has changed.
In order to get these events the line must be monitored with asnCtiLineMonitorStart

ArgumentType: AsnCtiNotifyLineInvalidPasswordChangedArgument

- PRIVATE -

AsnCtiNotifyLineInvalidPasswordChangedArgument is an Object

Name Type Optional Comment

u8sMonitorCrossRefID

string

CrossrefID for the corresponding asnCtiLineMonitorStart.

u8sLinePhoneNumber

string

Line identifier for the line where the changes happened.

invalidPasswordState

number

Indicates that the current password on the line is invalid (1) or is valid (0).

u8sUserName

string

The corresponding user name.
