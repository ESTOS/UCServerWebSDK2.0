asnConfigurationManagerSubscribeForeignConfig

subscribe to foreign config keys. Triggers query of these keys and will send asnConfigurationManagerForeignConfigUpdateEvent if somthing is found in the DB

ArgumentType: AsnConfigurationManagerSubscribeForeignConfigArgument

- PRIVATE -

AsnConfigurationManagerSubscribeForeignConfigArgument is an Object

Name Type Optional Comment

listKeys

UTF8StringList

list of keys (NO key prefixes!) to subscribe to / unsubscribe from

ResultType: AsnConfigurationManagerCRUDResult

- PRIVATE -

AsnConfigurationManagerCRUDResult is an Object

Name Type Optional Comment

iResultCode

number

ErrorType: AsnRequestError

AsnRequestError is an Object

Name Type Optional Comment

iErrorDetail

number

A number representing the error

u8sErrorString

string

A string representing the error
