asnRankingGetBestAgent

ermittelt den preferierten Agenten

Server ermittelt im Ranking-Verfahren den preferierten Agenten.
Entweder steht der preferierte Agent in der (preferred) Gruppe zur Verfügung.
Falls nicht, wird aus der Liste der Gruppen der bestAgent ermittelt.

ArgumentType: AsnRankingGetBestAgentArgument

- PRIVATE -

AsnRankingGetBestAgentArgument is an Object

Name Type Optional Comment

u8sGroupIDs

UTF8StringList

Liste von GroupIDs die berücksichtigt werden sollen (entweder GroupURI oder GroupName)

u8sPreferredAgentURI

string

yes

URI des bevorzugten Agenten (muss in der übergebenen u8sPreferredGroupID verfügbar sein)

u8sPreferredGroupID

string

yes

GroupURI in der der bevorzugte Agent verfügbar sein muss
