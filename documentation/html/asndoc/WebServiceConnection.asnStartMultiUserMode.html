asnStartMultiUserMode

Web service interfaceStart MultiUser mode on this connection.

This module defines strutures and operations for the web service.
This function is only used by the WebService, after it has established a connection to the UCServer.
After this function returned success, the ROSE headers include a SessionID parameter, and multiple user login sessions
can be multiplexed over this connection.
