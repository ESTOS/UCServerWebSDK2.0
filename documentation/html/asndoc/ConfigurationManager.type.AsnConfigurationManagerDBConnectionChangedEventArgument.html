AsnConfigurationManagerDBConnectionChangedEventArgument

update (UCWeb sends the changes from DB to UCServer)

- PRIVATE -

AsnConfigurationManagerDBConnectionChangedEventArgument is an Object

Name Type Optional Comment

bConnected

boolean

true: ucweb has connected to config db
false: ucweb has disconnected from config db

bFilledWithConfigurationSettings

boolean

additional info only if bConnected is true: Does the settings table contain settings ? (or is it empty)
